Above is the side-by-side comparison of the original design and my vision for the new design.

In my opinion the layout of the website is overloaded with the content. I found several changes that I could implement to make the navigation and content presented in a better way.

Visit original website
Visit re-designed website

Main navigation of the website (on the left side) has many categories, with sub-categories, making it look crowded and hard to navigate. In my proposed design I move some of the most important menu items on top of the page. This makes it more visible, and easier to navigate.

Visit original website
Visit re-designed website

The footer of the original site is only used as a decoration. In the new design I use bottom of the page for additional section of links that did not fit in primary navigation, and appear to look better when grouped with similar sections. In addition, location of the school, "About us", and social network links are now visible.

Visit original website
Visit re-designed website

Below a long navigation menu, in the original design, there is a list of archive records, grouped by month and a year. Archive consist of announcements made that month. There is absolutely no need for it to occupy that much space on the main page, and it should be a page on its own. That is why I included only the link to the archive on the front page.

Visit original website
Visit re-designed website

Now that navigation is relocated, it is time to re-design how main content is presented. Main content consist of announcements, school news, events, and planned activities. However, they are not separated in any visible way. Proposed design addresses these issues by grouping some content together, and some moving to a separate section.

Visit original website
Visit re-designed website

There are many photograph that are shared on the school's website to celebrate, and promote certain school services or students' achievements. By introducing an image slider within the main content section, I solve several problems: it saves space by eliminating white space , makes layout more organized and clean, and adds to more modern overall appearance.

The project is still on-going, I only worked on it in my spare time as I was learning how to built custom themes in WordPress. At the moment the front page is static, since I disabled all the dynamic content rendering until the project is ready.

This project is very important for my professional growth, since I have learned a lot about WordPress and improved my PHP skills. It also brings me joy knowing that one day, when I give this as a gift, I will be able to give back to my school.

Thank you!
